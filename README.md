# The dictionary editor 
A simple `YAML` and `JSON` editor written in Python.

Installation:
```
$ git clone https://github.com/dantard/dict-editor.git
$ cd dict-editor
$ pip install . 
```

Usage:
```
$ dict-editor filename.yaml
```
or

```
$ dict-editor filename.json
```
